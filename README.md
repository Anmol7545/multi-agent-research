# Multi-Agent Research Assistant

An AI-powered research assistant using LangChain, Groq LLM, and DuckDuckGo search.

## Tech Stack
- Python, LangChain, Groq LLM, FastAPI, DuckDuckGo Search

## Setup
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python src/main.py
```

## Features
- Real-time web search
- AI-powered summarization
- Interactive CLI interface
